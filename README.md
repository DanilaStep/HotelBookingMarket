# HotelBookingMarket
This application takes data from the server about the goods and displays them on the screen.
4 screens are implemented:
* hotel screen
* screen with list of numbers
* booking screen
* booking completion screen

# Stack:

 * Kotlin
 * MVVM
 * Multimodule
 * Koin
 * Navigation Component
 * Coroutines
 * StateFlow
 * Retrofit
 * Glide
